# Users
User and permission system scaffolds for the Reactor CMS.
 
---

## Installation
Installing Users is simple.
 
Pull this package in through [Composer](https://getcomposer.org).
```js
{
    "require": {
        "reactor/users": "dev-master"
    }
}
```
